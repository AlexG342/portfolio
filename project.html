Web Project

For this project we were asked to make a website from scratch for something we enjoy. I decided to make mine off of a game I really enjoyed. It's a simple indie game that I have been playing for quite some time and I noticed that they really don't have a website made for the game. So I decided to make a website that showcases the game a bit and tells you where you can purchase the game and as well as info on the game and developers.
